Project 2

Our team is working again on behalf of Forestview and their second wave of SafetyZone© proposals to key customers.

For this project, we were tasked with creating an application that simulated the user interface for a business of our choice. We decided to create the UI for employees at a hotel front desk and concierge with the following criteria:

  • Using Python3, create a 'robust' application that simulates a business' user interface.
  • Use a GUI to allow user to fulfill the proposed duties of the application.
  • Store, collect, and searche data sets using Json files to simulate inventory.
  • Use Doc strings throughout for readability of the code.
  • Package the application to an executable that users can download from this webpage (after signing the user agreement).

As before, we chose tkinter and Pylint as the required libraries.

Specifics

Our application simulates a hotel front desk, and includes the following functionalities:

  • Guest check in and check out - with room availability visualized for ease of use.
  • Room service - from both in-house kitchen and front desk pantry.
  • Recipe Builder - easy to add new recipes to the menu
  • Guest lookup - search for guests by name

We strive to provide quality service, and scalable, cyber-secure solutions.
